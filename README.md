# ProtoCreator

Unity中用UIToolKit做界面的一个编辑Proto以及生成C#文件的工具
支持新增、删除、保存Proto 文件，以及通过cmd生成C#文件

导入Editor后，在菜单栏的tool/ProtoCreator 可以看到
